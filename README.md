# TFG_DilithiumFinalVersion

This is a repository starting with the basic Dilithium version of [TFG](https://github.com/AriadnaPrat/TFG)
The basic Dilithium scheme is interactive, and the goal of this repository is to get a non-interactive Dilithium scheme with Fiat-Shamir. Our code will be based on Lyuvashesvky's paper.

Reference: https://eprint.iacr.org/2024/1287.pdf
